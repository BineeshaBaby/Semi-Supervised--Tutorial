Introduction

Semi-supervised learning is a powerful machine learning paradigm that combines the strengths of supervised and unsupervised learning techniques. It is particularly useful when labeled data is scarce but unlabeled data is abundant.

In this tutorial, we will explore two popular methods:

  • Self-Training: A technique that iteratively labels unlabeled data based on the predictions of a trained model.
  • Label Propagation: A graph-based algorithm that spreads labels to unlabeled data points based on similarity.
